# Required intake questions

Ask for these inputs at the start unless the user already supplied them.
Keep the questions short and conversational.

1. What is your website URL?
2. What photography genre are we building this for?
3. What exact offer should the quiz help them claim?
4. Do you want the title to feel like a challenge, such as `Can You Get 4/4 Right?`
5. What top objections, myths, or hesitations should the quiz address?
6. What top features, benefits, products, or differentiators should we highlight?
7. Is there anything you do not want mentioned?
8. Paste any extra context, FAQs, sales language, or notes you want used.

## Notes

- Website URL is required whenever possible.
- The offer is required whenever possible. Do not quietly substitute a generic consultation CTA if the offer is unknown.
- If the user does not know their objections, infer them from the genre and website after noting that you are making a best effort.
- If the user wants a draft before giving the offer, use a clearly labeled placeholder offer and mark it in the input summary.
- The main 4 quiz questions should usually feel like answerable challenge questions, not preference polling.
