# Build Funnel

Use this when the user asks to build a funnel, campaign, lead path, quiz,
booking flow, reactivation flow, ad-to-page path, or broader studio marketing
system.

## Route The Work

- Use `funnel-design` for simple website inquiry paths and page-level CTA flow.
- Use `photo-studio-website-copywriter` for landing page and website copy.
- Use `boudoir-copywriter` for boudoir funnel copy.
- Use `typeform-quiz-popup-builder` for Typeform quiz popups.
- Use `google-ads-photographers` for Google Search campaign assets and
  ad-to-landing-page alignment.
- Use `studio-reactivation` for old lead or dead CRM campaigns.
- Use `studio-booking-event` for in-studio preview nights or open-house booking
  events.
- Use `studio-gtm-strategy` only for a full end-to-end marketing system with
  offer, ads, quiz, follow-up, phone script, and measurement.
- Use `site-builder` and `supabase-cms` only when the funnel needs to be
  implemented in this Next/Supabase site.

## Operating Principles

- Pick the smallest funnel that solves the stated problem.
- Do not run the full GTM process for a simple page CTA or quiz request.
- Start from real proof, real offers, real locations, and real constraints.
- Never invent testimonials, results, guarantees, prices, awards, or scarcity.
- Search current platform documentation when campaign, tracking, Typeform,
  Google Ads, Supabase, or deployment behavior is uncertain.
- Keep human-facing drafts easy to review. For substantial copy/strategy, write
  clean Google-Doc-ready Markdown and create a Google Doc when that connector is
  available and the user wants a shareable review artifact.

## Output

When finished, report:

- Funnel type
- Primary CTA and conversion path
- Copy/assets produced
- Implementation needed, if any
- Tracking or platform setup notes
- Open proof/offer questions
