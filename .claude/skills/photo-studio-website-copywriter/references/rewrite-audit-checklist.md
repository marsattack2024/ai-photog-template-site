# Rewrite and Audit Checklist

Use this for existing website reviews.

## Audit Output Structure

# Website Copy Audit: [Studio]

## Bottom Line
One direct paragraph explaining the biggest conversion issue and highest-leverage fix.

## What Is Working
List only real strengths.

## What Is Leaking Conversions
Group by:
- Hero clarity.
- Genre confusion.
- CTA friction.
- Proof gaps.
- Objection gaps.
- Mobile friction.
- Form friction.
- StoryBrand misalignment.
- Offer or funnel mismatch.

## Priority Fixes
Rank fixes by likely impact:
1. Highest leverage.
2. Next best.
3. Nice-to-have.

## Rewritten Copy Blocks
Provide improved copy for the weakest sections.

## Proof Needed
List specific proof the studio should collect.

## Implementation Notes
Include CTA placement, form placement, internal links, tracking notes, and page structure changes.

## Scoring Rubric

Score each 1-5:
- Hero clarity.
- Emotional relevance.
- Genre specificity.
- CTA clarity.
- Proof strength.
- Objection handling.
- Mobile readability.
- Revenue Formula alignment.

Verdict:
- Ready to publish.
- Publish after proof is added.
- Needs rewrite before traffic.
