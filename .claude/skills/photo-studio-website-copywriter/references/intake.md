# Intake

Use this intake for full websites, landing pages, and funnel-connected copy.

## Full Intake

1. Studio name and location.
2. Service area.
3. Genre or genres.
4. Primary page or pages needed.
5. Ideal client.
6. Clients to attract.
7. Clients to repel.
8. Offer structure.
9. Session fee, IPS/product model, and average sale for strategy.
10. Leads per month.
11. Booking rate.
12. Biggest bottleneck: leads, booking rate, average sale, or follow-up.
13. Current marketing channels.
14. Destination page or funnel.
15. Existing proof.
16. Voice preference.
17. Main CTA.
18. Technical notes: CRM, scheduler, quiz, tracking, thank-you page.

## Small Ask Intake

For one short section, only ask for:
- Studio name/location.
- Genre.
- Section/page goal.
- CTA.

## Best-Effort Rule

When the user has already provided enough context to draft, do not stall with excessive questions. Draft with clear assumptions and TODO markers.
