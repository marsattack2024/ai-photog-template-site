# Ad Platform Playbook

## What this file is for
Use this file when the request involves paid ads, ad-set copy packages, or campaign deliverables. It captures structure, output counts, formatting rules, and campaign-level thinking pulled from the studio ad-copy guide.

## Core campaign rules
- Each campaign should focus on one genre or one boudoir offer at a time.
- Ad copy should align with the landing page promise and tone.
- The objective should be clear: leads, bookings, consultation applications, awareness, retargeting, or a special campaign.
- Customer persona should shape the angle, not just demographics.
- Copy should be emotionally resonant, but still concise and usable.

## Style and formatting rules
- warm, friendly, supportive, professional
- straightforward and relatable language
- no excessive punctuation or full capitalization
- title case for headlines
- sentence case for descriptions and body text unless platform convention requires otherwise
- no competitor comparisons or unverifiable claims

## Meta ad deliverable default counts
When the user asks for a “full ad set” and does not specify counts, use this default package:
- 8 headline variations
- 5 description variations
- 8 primary text variations

If the user asks for fewer, obey the user.
If the user asks for more, scale up.

## What strong Meta headlines should do
Across a set of headlines, aim to cover multiple jobs:
- direct benefit
- objection relief
- local relevance if provided
- urgency when appropriate
- testimonial-style proof
- process-ease framing
- story / transformation curiosity

## What strong descriptions should do
- reinforce the offer or emotional benefit
- keep urgency concise when relevant
- mention specifics or offer details when available
- end with studio name and location if the user requests that format

## What strong primary texts should do
Across a full set, vary these angles:
- fear/hesitation acknowledgment
- confidence/identity outcome
- practical reassurance
- feature-to-benefit explanation
- testimonial-style proof
- client story mini-narrative
- urgency / limited availability
- milestone or permission-based angle

## Google ad rules
When writing Google ads, default to:
- keyword and intent alignment
- local relevance
- benefits that reduce friction
- privacy and process clarity
- compliance and realism

Useful headline themes:
- Private Boudoir Sessions In [City]
- Guided Boudoir Photography
- Hair And Makeup Included
- Boudoir For Real Women
- Book Your Consultation

Useful description themes:
- private studio, guided posing, wardrobe help, confidence-building, discreet experience, consultation, reveal process

## Landing-page alignment rule
For any ad set, make sure the copy and page promise align on:
- who it is for
- what is included
- emotional promise
- offer or campaign framing
- CTA language

The ad should not make the page feel like a bait-and-switch.

## Creative pairing rules
- body-confidence message -> use body-diverse imagery or testimonial proof
- process-made-easy message -> BTS or step-by-step visuals
- privacy message -> calm, discreet, elegant visuals rather than overtly provocative imagery
- gift message -> product or album visuals can strengthen the offer
- review-proof message -> review screenshots, testimonial clips, UGC, or green-screen proof concepts

## Campaign-level considerations to respect in strategy requests
These items are useful when the user asks for campaign strategy, launch checklists, or handoff notes:
- Meta account setup and permissions
- pixel / event tracking
- form functionality and thank-you-page flow
- CRM automations and immediate follow-up
- retargeting before or alongside cold traffic
- TOF ads built around the studio USP
- custom audiences and lookalikes when available
- quiz ads when quiz funnels exist
- video ads and testimonial ads in every campaign mix
- UTM parameters for tracking
- exclusion lists
- mobile optimization
- multiple copy variants to avoid fatigue
- policy compliance and privacy language

## Special campaign prompts this skill should handle well
- model calls
- 40 over 40
- birthday campaigns
- holiday campaigns
- anniversary gift campaigns
- bridal boudoir
- confidence reset / reinvention campaigns

## Compliance notes
- do not promise impossible outcomes
- do not imply negative personal attributes in a way that violates platform policies
- do not write copy that sounds creepy, male-gaze, or objectifying
- keep the tone supportive and client-centered
- when in doubt, describe the experience and proof rather than making extreme claims
