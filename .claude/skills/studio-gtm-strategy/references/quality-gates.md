# Quality Gates — All Stages

Run before presenting each artifact. If any fails, revise or ask.

## Stage 1 — Studio Discovery
- [ ] Genre identified (primary + complementary)
- [ ] Find Your Stage diagnosis (1-5)
- [ ] Revenue Formula baseline calculated
- [ ] Website audited (or noted "no website")
- [ ] Current messaging extracted
- [ ] Top 2-3 competitors scanned
- [ ] Genre objections pulled and confirmed
- [ ] Social proof inventory catalogued
- [ ] IPS vs digital delivery confirmed
- [ ] Platform recommendation matches genre demand type
- [ ] Budget tier assigned

## Stage 2 — USP & Offer
- [ ] USP excavation completed (not skipped)
- [ ] "So What?" test on all differentiators
- [ ] USP validated against competitor gaps
- [ ] USP is specific (not "passionate about photography")
- [ ] Session fee defined as door-opener
- [ ] IPS confirmed or flagged for implementation
- [ ] Experience stack documented
- [ ] Door-opener offer defined
- [ ] Offer Math calculated with Revenue Formula
- [ ] Offer name created
- [ ] No unsubstantiated claims
- [ ] Next seasonal campaign identified

## Stage 3 — Messaging
- [ ] Positioning statement passes Scroll Test
- [ ] USP woven into every messaging level
- [ ] Buyer persona matches genre
- [ ] 3 AM Thoughts exercise completed (5+)
- [ ] Voice personality: 3 adjectives with do/don't
- [ ] All objections have reframes + studio answer + copy + story proof + quiz angle
- [ ] Messaging couldn't be pasted onto a competitor's site
- [ ] No invented claims
- [ ] Context Handoff Block generated

## Stage 4 — Ad Copy
- [ ] Hook ≤ 12 words
- [ ] First 125 chars standalone
- [ ] Every ad connects objection to studio USP
- [ ] Specific proof included (or TODO)
- [ ] Single CTA per ad
- [ ] Objection-Flipper: one per top 5 objections
- [ ] Video Testimonial template/copy provided
- [ ] Quiz/Door-Opener ads promote voucher
- [ ] Retargeting carousel: 10 slides
- [ ] Retargeting copy: 3 audiences
- [ ] Hook Bank: 5 variations for top 2 angles
- [ ] Google RSA if timeline genre
- [ ] Angle count matches budget tier
- [ ] No fake scarcity
- [ ] Voice is warm, not bro-marketer

## Stage 5 — Quiz & Landing Page
- [ ] Quiz: 3-5 question/statement pairs
- [ ] Each question addresses real genre objection
- [ ] Each statement 150+ words with studio USP
- [ ] Fair Enough slide included
- [ ] Quiz title matches ad messaging
- [ ] Voucher amount consistent across all assets
- [ ] Landing page headline echoes ad hook
- [ ] Landing page: all 9 sections
- [ ] Single CTA throughout
- [ ] Mobile-first responsive
- [ ] Social proof real or marked TODO
- [ ] FAQ uses Genre Objection Map copy
- [ ] Installation notes included

## Stage 6 — Follow-Up
- [ ] Call-Text-Call fires within 5 minutes
- [ ] Thank-you page: video, scheduler, testimonial, investment context
- [ ] Pre-call emails: 6 messages, 7 days
- [ ] SMS: 4 messages, 8 days
- [ ] No links in first texts
- [ ] Insight email teaches something valuable
- [ ] 2+ different client stories used
- [ ] Breakup message in both email and text
- [ ] Nurture track: 4 emails, value-first
- [ ] Reactivation play documented
- [ ] CRM pipeline stages defined
- [ ] Single CTA per message
- [ ] Merge fields marked
- [ ] Voice is warm, not salesy

## Stage 7 — Phone Script & Booking
- [ ] Pre-call ritual: 7 items
- [ ] All 5 script stages present
- [ ] Discovery: 6+ genre-specific questions
- [ ] Future-pacing questions included
- [ ] Prescription references offer by name + IPS context
- [ ] 4 objection scripts adapted to this studio
- [ ] Standard + Prestige Close
- [ ] Booking confirmation: date, prep, questionnaire, reschedule
- [ ] Reminder: 4 messages (7-day, 24h, 2h, no-show)
- [ ] Warm, consultative tone
- [ ] Investment range stated clearly
- [ ] Close offers two specific dates

## Stage 8 — Summary
- [ ] Revenue Formula calculated (current + projected)
- [ ] Math realistic for genre/market
- [ ] All stages covered in blueprint
- [ ] Checklist photography-specific
- [ ] KPIs match studio benchmarks
- [ ] End-to-end test step included
- [ ] Phone script practice step
- [ ] Video testimonial collection step
- [ ] Reactivation + organic included
- [ ] Launch over perfection emphasized
