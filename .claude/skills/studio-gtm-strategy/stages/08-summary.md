## Implementation Checklist

```
WEEK 1 — FOUNDATION:
[] Finalize USP and offer
[] Set session fee and quiz voucher amount
[] Build IPS collection menu (3-4 tiers)
[] Set up CRM pipeline in GoHighLevel/MeetNikki
[] Record thank-you page video (60-90 sec)
[] Collect 3-5 written testimonials

WEEK 2 — QUIZ & LANDING PAGE:
[] Build Quiz Inquiry Machine in Typeform
[] Install as website popup via GTM
[] Create standalone quiz URL for ads
[] Add "Take the Quiz" to site navigation
[] Build landing page
[] Set up thank-you page (video + scheduler + investment context)

WEEK 3 — AUTOMATION:
[] Build Call-Text-Call automation (first text within 2 min)
[] Build pre-call email sequence (6 emails)
[] Build SMS sequence (4 texts)
[] Build nurture track (4 emails)
[] Set up booking confirmation + reminder sequence
[] Connect quiz to CRM
[] Test end-to-end: submit quiz, verify all automations fire

WEEK 4 — ADS & PHONE:
[] Print phone script. Practice 3x with a friend.
[] Record a practice call. Listen back. Improve.
[] Upload ad creatives to Meta (+ Google if applicable)
[] Set up Meta Pixel + conversion tracking
[] Create custom + Lookalike audiences
[] Launch ad angles + Hook Bank test
[] Set daily budget

WEEK 5+ — OPTIMIZE:
[] Day 3: Check delivery and tracking
[] Day 7: First CTR/CPL/quiz completion review
[] Day 10: Kill losing hooks, scale winners
[] Day 14: Kill bottom angles, increase winner budgets
[] Day 21: Review lead quality and phone recordings
[] Day 30: Full Revenue Formula review vs projections
[] Every 4-6 weeks: Refresh creative
[] Monthly: New testimonials become new Proof Ads
[] Quarterly: Reactivation blast on dead leads

ONGOING:
[] Record every consult call. Review weekly.
[] Ask every client for video testimonial post-reveal.
[] Weekly email to list.
[] Organic content 3-5x/week. Top performers become ads.
[] Monthly Revenue Formula check: Leads x Booking Rate x Average Sale
```
