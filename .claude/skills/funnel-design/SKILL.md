# Photography Funnel Design

Use this when deciding how a photography studio website should turn visitors
into qualified inquiries. This is not a SaaS funnel, agency funnel, or course
sales funnel.

## Core Rule

Brochure sites should stay simple. Prefer a clear inquiry path over complex
multi-step automation unless the brief explicitly calls for it.

## Funnel Types

- Standard studio site: page CTA -> inquiry form -> thank-you page -> manual or
  CRM follow-up
- Campaign landing page: ad/organic traffic -> single offer page -> short form
  -> confirmation -> consultation or availability check
- Lead magnet: guide/checklist/style quiz -> email capture -> nurture -> session
  inquiry
- Gallery-led flow: gallery proof -> service context -> inquiry
- Referral/vendor flow: venue/vendor/social traffic -> trust proof -> inquiry

## Design Process

1. Identify the primary booking action: inquiry, call, consult, package request,
   availability check, or lead magnet.
2. Define the visitor's likely hesitation by genre.
3. Choose one primary CTA and one secondary CTA at most.
4. Map the path from first click to post-submit expectation.
5. Decide what data the form must collect. Keep it short unless qualification is
   more important than volume.
6. Specify proof needed near the CTA: testimonials, galleries, reviews, privacy
   note, pricing anchor, or process clarity.

## Deliverable

Return:

- Audience and offer
- Traffic sources
- Primary CTA and secondary CTA
- Page-by-page conversion path
- Form fields and why each is needed
- Thank-you page message
- Follow-up recommendation
- Measurement events to track
- Risks or assets needed before launch

## Recommended Form Fields

Default inquiry:

- Name
- Email
- Phone optional unless calls are core to sales
- Service interest
- Date or timing window when relevant
- Message or "what are you dreaming up?"

Boudoir may add privacy-safe intent fields. Wedding may add venue and date.
Family/newborn may add due date or preferred season.

## Measurement

Track at minimum:

- CTA click
- Form start if possible
- Inquiry submit
- Thank-you page view
- Gallery or pricing section engagement when relevant
