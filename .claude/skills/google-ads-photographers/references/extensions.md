# Google Ads Extensions

Extensions must reinforce the same search intent as the campaign. Do not dump generic studio links into every campaign.

## Callouts

Create 6 callouts. Each must be 25 characters or fewer.

Good callout categories:
- Posing Guidance
- Hair & Makeup
- Private Studio
- Client Closet
- Wardrobe Help
- Safe Newborn Studio
- Family Styling Help
- Same-Day Reveal
- Heirloom Albums
- Wall Art Design
- Free Consultation
- Local Studio
- Fully Guided Session

Do not use unverifiable claims.

## Structured Snippets

Use at least one header with at least 3 values. Use the most relevant headers:

- Services
- Styles
- Amenities
- Products
- Locations
- Session Types
- Specialties

Examples:

Boudoir amenities:
- Hair & Makeup
- Posing Guidance
- Client Closet
- Private Reveal

Newborn services:
- Newborn Portraits
- Maternity Photos
- Baby Milestones
- Family Portraits

Branding services:
- Headshots
- Team Photos
- Brand Portraits
- LinkedIn Images

## Sitelinks

Create 8 sitelinks. Each sitelink needs two description lines, each 35 characters or fewer.

Default sitelink set:

1. Portfolio
2. The Experience
3. About The Studio
4. Testimonials
5. FAQ
6. Book A Consultation
7. Client Resources
8. Contact The Studio

Studio conversion standard:
- Do not create a public "Pricing" or "Investment" sitelink by default.
- Use consult, FAQ, experience, or product education instead.
- If the user explicitly requires a pricing/investment sitelink, include a risk note.

Destination rules:
- Use the matching genre page when available.
- Use placeholders if URLs are not provided.
- Never send a genre campaign to a mixed-gallery page when a dedicated page exists.

## Promotion Extension

Use only when there is a real offer.

Include:
- promotion name
- value or incentive
- terms
- date range if available
- final URL

Examples:
- Style Quiz Credit
- Application Campaign
- Complimentary Consultation
- Print Credit
- Waived Session Fee

Do not invent discounts or values.

## Image Extension Direction

Do not choose images unless the user provides assets. Provide direction:

- Use clean, high-quality images with strong subject focus.
- Match the campaign genre.
- Avoid overly suggestive boudoir images that may trigger policy issues.
- Show real clients only with consent.
- Use diverse body types/ages when relevant.
- Test close portraits, behind-the-scenes, product/album images, and studio environment.

## Lead Form Extensions

Use lead form extensions only when:

- The studio can respond quickly.
- CRM integration exists.
- The offer is simple enough to understand without a full landing page.

For higher-ticket IPS studios, prefer landing-page conversion or consultation scheduling so the prospect gets enough context before inquiry.
