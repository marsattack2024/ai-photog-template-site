# Genre Keyword Strategy

Use one search campaign per genre or intent cluster. Keep each campaign narrow enough that every keyword, RSA asset, extension, and landing page can speak to the same buyer need.

## Default Keyword Pattern

Build around:

- `[city] [genre] photographer`
- `[genre] photographer near me`
- `[genre] photography [city]`
- `[genre] photography studio`
- `[genre] photoshoot`
- `professional [genre] photographer`
- `local [genre] photographer`
- `[genre] portraits`
- `[nearby area] [genre] photographer`
- `[genre] session [city]`

Use 10 keywords by default. Favor phrase and exact match for small budgets.

## Match Type Guidance

- **Exact match:** highest control; use for core bottom-of-funnel terms.
- **Phrase match:** good for local service discovery while staying relevant.
- **Broad match:** use only when conversion tracking is clean and the account has enough data/budget.

## Negative Keyword Starters

Use only if relevant:

- free
- cheap
- diy
- jobs
- career
- salary
- classes
- course
- school
- camera
- equipment
- png
- stock photo
- ideas
- poses
- pinterest
- template
- app
- ai
- passport
- walmart
- jcpenney
- sears
- craigslist

Add genre-specific negatives. For example, headshot campaigns may need to exclude passport photos; newborn campaigns may need to exclude hospital jobs; wedding campaigns may need to exclude venues if intent is wrong.

## Boudoir

Buyer intent:
- Wants privacy, guidance, confidence, hair/makeup, wardrobe, and reassurance.

Keyword themes:
- boudoir photographer [city]
- boudoir photography near me
- boudoir studio [city]
- luxury boudoir photographer
- bridal boudoir photographer
- women boudoir photography
- private boudoir session
- empowerment photography [city]

Copy angles:
- private studio
- guided posing
- professional hair and makeup
- size-inclusive wardrobe
- albums and wall art
- all-female team if verified
- consultation before booking

Avoid:
- oversexualized language
- unverified privacy/team claims
- implying ideal body type
