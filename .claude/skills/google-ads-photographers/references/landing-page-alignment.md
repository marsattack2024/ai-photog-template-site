# Landing Page Alignment

A Google Ads campaign converts when the search term, ad promise, and landing page match.

## Minimum Landing Page Checks

For every campaign, verify:

- The page is genre-specific.
- The page headline reflects the same service and location as the ad.
- The first CTA matches the ad CTA.
- The page answers the top buyer objections.
- The page includes relevant proof.
- The form is short and easy on mobile.
- The thank-you page has a unique URL or tracked event.
- The page loads quickly on mobile.
- The images match the genre and offer.
- The offer terms are consistent with the ad.

## Page Content by Genre

Boudoir:
- privacy
- posing guidance
- wardrobe/client closet if verified
- hair/makeup if verified
- albums/wall art
- proof from real women
- consultation CTA

Newborn:
- safety
- timing window
- baby-led pacing
- parent reassurance
- studio comfort
- maternity/newborn bundle only if real

Maternity:
- flattering posing
- wardrobe/gowns if verified
- timing guidance
- hair/makeup if verified
- partner/family inclusion if offered

Family:
- kids do not have to behave perfectly
- styling help
- relaxed session flow
- artwork for home
- seasonal/session availability

Branding/headshots:
- professional credibility
- posing help
- usage examples: LinkedIn, website, press, team page
- turnaround if verified
- team/session options

## CTA Alignment

Ad CTA and page CTA should match.

Good:
- Ad: "Schedule Your Consultation"
- Page button: "Schedule My Consultation"

Weak:
- Ad: "Claim Your Credit"
- Page button: "Browse Gallery"

## Tracking Requirements

Recommend:

- Thank-you page URL for form submissions.
- Google Ads conversion action.
- Google Tag Manager or direct tag installation.
- GA4 event for lead, booked call, purchase/deposit if relevant.
- UTM parameters on all final URLs.
- CRM integration test.
- Test form submission from mobile before launch.

## UTM Template

Use when appropriate:

`utm_source=google&utm_medium=cpc&utm_campaign={campaignid}&utm_content={creative}&utm_term={keyword}`

If the client has an existing naming convention, use that instead.
